# Data Cleaning using Python

## Objective
Clean and preprocess a messy dataset to make it analysis-ready by handling missing values, duplicates, inconsistent formatting, outliers, and incorrect data types.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Key Features
- Data quality assessment
- Missing value treatment
- Duplicate removal
- Data standardization
- Outlier detection using IQR
- Data type correction
- Before vs. After comparison
- Export cleaned dataset

## Outcome
Prepared a clean, consistent, and analysis-ready dataset suitable for further analysis and machine learning.
